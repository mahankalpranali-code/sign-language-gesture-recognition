# Sign Language Gesture Recognition System

## Overview
This project is an AI-based real-time Sign Language Gesture Recognition System developed using MediaPipe, TensorFlow, and Keras.

The system detects hand gestures through a webcam and predicts sign language gestures using a trained deep learning model.

---

## Features
- Real-time hand tracking
- Gesture prediction using deep learning
- MediaPipe hand landmark detection
- TensorFlow/Keras model integration
- Webcam-based recognition system

---

## Technologies Used
- Python
- TensorFlow
- Keras
- MediaPipe
- OpenCV
- NumPy
